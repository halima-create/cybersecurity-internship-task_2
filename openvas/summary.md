# OpenVAS Vulnerability Summary

## Critical Issues:
- vsFTPd 2.3.4 backdoor vulnerability
- Apache outdated version with multiple CVEs
- MySQL weak authentication
- OpenSSH outdated and vulnerable to several exploits

## High Severity:
- PHP vulnerabilities in DVWA and Mutillidae
- SMB vulnerabilities (MS08-067, MS17-010)

## Medium:
- SSL certificate issues
- Missing security headers

## Low:
- Informational misconfigurations

## Conclusion:
Metasploitable2 contains numerous critical weaknesses suitable for penetration testing learning.
