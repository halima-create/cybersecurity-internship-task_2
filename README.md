# Network Reconnaissance & Scanning 

## Reconnaissance
- Performed passive information gathering using Whois, Nslookup, search queries, and basic host discovery.
- Ran active checks such as ping sweep and banner grabbing on the target network.

## Nmap Scanning
- Conducted TCP SYN scan (-sS)
- Checked service versions (-sV)
- Identified OS fingerprinting (-O)
- Collected open ports and active services on the Metasploitable2 machine.

## Service Enumeration
- Tested FTP anonymous access.
- Accessed the HTTP interface and identified applications like DVWA, Mutillidae, phpMyAdmin, and WebDAV.
- Captured simple service responses using curl and nc.

## Traffic Analysis
- Observed FTP and HTTP traffic behavior in Wireshark.
- Noted characteristics of unencrypted protocols.

## Additional Notes
- Basic interactions performed with exposed services.
- Initial steps toward vulnerability analysis and firewall rule testing.
